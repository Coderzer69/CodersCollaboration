# 🚀 Coders Collaboration

A real-time collaborative code editor that enables multiple developers to write and edit code together simultaneously. Built with **React**, **Monaco Editor**, **Yjs**, **Socket.IO**, and **Express**, the application provides live synchronization, presence awareness, and a modern coding environment.

## ✨ Features

- 👨‍💻 Real-time collaborative code editing
- ⚡ Instant synchronization across multiple users
- 👥 Live user presence (see connected collaborators)
- 🎨 VS Code-like editing experience using Monaco Editor
- 🔄 Conflict-free synchronization powered by Yjs (CRDT)
- 🌐 Socket.IO based real-time communication
- 📱 Responsive and clean UI
- 🚀 Fast frontend powered by Vite

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Coderzer69/CodersCollaboration-.git
```

```bash
cd CodersCollaboration-
```

---

### 2. Install Backend Dependencies

```bash
cd Backend
npm install
```

---

### 3. Install Frontend Dependencies

```bash
cd ../Frontend
npm install
```

---

## ▶️ Running the Project

### Start Backend

```bash
cd Backend
npm start
```

or

```bash
npm run dev
```

Backend runs on

```
http://localhost:3000
```

---

### Start Frontend

```bash
cd Frontend
npm run dev
```

Frontend runs on

```
http://localhost:5173
```

---

## 💻 Usage

1. Open the application in your browser.

2. Enter your username.

3. Open another browser window (or share the URL).

4. Join using a different username.

5. Start coding together in real time.
